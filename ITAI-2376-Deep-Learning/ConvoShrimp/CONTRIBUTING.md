# Contributing to ConvoShrimp

Thank you for your interest in contributing to ConvoShrimp.

## Development Setup

1. Fork the repository
2. Clone your fork: `git clone https://github.com/yourusername/convoshrimp.git`
3. Install dependencies: `npm install`
4. Create a branch: `git checkout -b feature/your-feature`
5. Make your changes
6. Run the dev server to test: `npm run dev`
7. Commit your changes: `git commit -m "Add your feature"`
8. Push to your fork: `git push origin feature/your-feature`
9. Open a Pull Request

## Code Style

- Use functional React components with hooks
- Follow existing naming conventions
- No native emojis - use Lucide icons instead
- Keep components focused and reusable

## Project Structure

- `/src/components/` - React components
- `/src/data/` - Static data and content
- `/public/` - Static assets

## Reporting Issues

Please use GitHub Issues to report bugs or request features. Include:

- Steps to reproduce
- Expected behavior
- Actual behavior
- Screenshots if applicable
