What I Learned — Deep Learning

Neural Network Fundamentals

How neurons compute output = activation(W·x + b). Weights scale input importance, biases shift activation thresholds. Forward propagation passes data through layers; backpropagation computes gradients to update weights via loss functions like CrossEntropyLoss.

Activation Functions

ReLU: f(x) = max(0, x) — sparse, fast, solves vanishing gradients. Sigmoid: squashes to (0,1) for probability. Tanh: centered at zero for balanced gradients. Without activation, stacked layers collapse to one linear transformation.

CNN Architecture

Convolution (3×3 kernels → feature maps) → Max Pooling (spatial reduction) → Batch Normalization (training stability) → Dropout (regularization) → Flatten → Dense Layers → Softmax (class probabilities). Studied evolution: LeNet → AlexNet → VGGNet → GoogLeNet → ResNet → EfficientNet.

Generative Models — Diffusion

Denoising Diffusion Probabilistic Models (DDPMs): forward process adds Gaussian noise over T steps; reverse process trains a U-Net to denoise one step at a time, generating images from pure random noise. Noise scheduling controls training stability.

Sequence Models & Attention

Encoder-decoder architectures for variable-length sequences. Attention mechanisms let the decoder focus on relevant encoder positions instead of compressing input into a fixed vector — the foundation of Transformers.

PyTorch Implementation

Built CNN class (nn.Module), training loops with DataLoader, data augmentation (RandomCrop, HorizontalFlip), Adam optimizer + StepLR scheduler, GPU acceleration, trained on CIFAR-10 (60K images, 10 classes).
