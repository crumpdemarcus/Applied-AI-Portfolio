### Deep Learning (ITAI-2376)

Interactive applications that visualize CNN architectures, activation functions, and neural network fundamentals — from a React app to a hand-illustrated web comic.

#### [ConvoShrimp](./ITAI-2376-Deep-Learning/ConvoShrimp/)
A React application that visualizes CNN architecture through a biological analogy with the mantis shrimp visual system. Users walk through convolution, pooling, feature extraction, and classification via an animated, scroll-driven interface.

**Key Features:**
- Animated CNN architecture walkthrough with biology-to-neural mapping
- Embedded PyTorch code examples with syntax highlighting
- Interactive timeline of CNN evolution (LeNet → EfficientNet)
- One-click PDF export of the full presentation

**Technologies:** React 19, Vite 7, TailwindCSS, Framer Motion, PyTorch

---

#### [Neural Network Squad](./ITAI-2376-Deep-Learning/Neural-Network-Squad/)
A browser-based interactive comic that teaches weights, biases, and activation functions through original illustrated characters and live JavaScript demos — zero dependencies, runs in any browser.

**Key Features:**
- Five character-driven explainers for ReLU, Sigmoid, Tanh, weights, and biases
- Interactive sliders that adjust values in real time
- Fully responsive, zero-dependency vanilla HTML/CSS/JS
- Designed for beginner audiences while keeping ML vocabulary accurate

**Technologies:** HTML5, CSS3, JavaScript

---

#### [Arrival](./ITAI-2376-Deep-Learning/Arrival/)
Formal deep learning deliverables — a final presentation deck and technical lab report exploring neural network concepts through an *Arrival*-inspired theme.

**Key Features:**
- Technical presentation design for deep learning topics
- Research-style lab report with analysis and findings
- Interdisciplinary application of neural network concepts

**Technologies:** Research documentation, presentation design

---

#### [Lab 08 — Diffusion Models](./ITAI-2376-Deep-Learning/L08_Diffusion_DeMarcus_Crump_ITAI_2376.pdf)
A 89-page Jupyter notebook covering denoising diffusion probabilistic models (DDPMs) — from the forward noising process through reverse denoising and image generation.

**Topics Covered:**
- Forward/reverse diffusion process and noise scheduling
- U-Net architecture for denoising
- Image generation from random noise

**Technologies:** Python, PyTorch, Diffusion Models, Jupyter Notebooks
